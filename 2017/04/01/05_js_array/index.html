数组属性

属性 描述
constructor 返回创建数组对象的原型函数。
length 设置或返回数组元素的个数。
prototype 允许你向数组对象添加属性或方法。

方法

方法 描述 备注
contat() 连接数组，并返回结果，不改变原数组
copyWithin() 在数组内拷贝指定元素到指定位置，改变原数组 ES6
every() 检测数组是否满足某个条件（函数），返回布尔类型，不改变原数组
fill() 用指定值替换数组中指定位置的元素，改变原数组 ES6
filter() 检测数组返回满足条件的所有元素，不改变原数组
find() 返回第一个满足条件的元素，无满足返回-1，不改变原始值 ES6
findIndex() 返回第一个满足条件元素的位置，无满足返回-1，不改变原始值 ES6
forEach() 调用数组的每个元素，并将元素传递给回调函数。
indexOf() array.indexOf(item,start) 从指定位置开始返回指定元素第一个出现的位置，无返回-1
join() 把数组的所有元素放入一个字符串。
lastIndexOf() 从指定位置从后向前查找返回第一个指定元素位置，无返回-1
map() 通过指定函数处理数组的每个元素，并返回处理后的数组。不改变原始值
pop() 删除数组的最后一个元素并返回删除的元素。改变原始值
push() 向数组的末尾添加一个或多个元素，并返回新的长度。改变原始值
reduce() 将数组元素计算为一个值（从左到右）。
reduceRight() 将数组元素计算为一个值（从右到左）。
reverse() 倒序数组，改变原始值
shift() 把数组的第一个元素从其中删除，并返回第一个元素的值。改变原始值
slice() 返回选取位置的所有元素，不改变原始值
some() 检测数组中是否有满足条件（函数）的值，返回布尔类型，不改变原始值
sort() 对数组的元素进行排序。改变原始值
splice() 插入、删除或替换数组的元素。
toString() 把数组转为字符串，逗号隔开，不改变原始值
unshift() 向数组的开头添加一个或更多元素，并返回新的长度。改变原始值

keys() 返回 A new Array iterator object.
values() 返回 A new Array iterator object.

数组去重

var isFunction = function (fn) {
    return Object.prototype.toString.call(fn) === '[object Function]';
}

Array.prototype.distinct = function (keyThunk) {
    var arr = this;
    var dic = {};
    var result = [];

    keyThunk = isFunction(keyThunk) ? keyThunk : function (val) {
        return typeof (val) + val;
    }

    var hasUnderfined = false;
    var hasNull = false;
    for (var i = 0; i < arr.length; i++) {
        var type = typeof (arr[i]);

        if (arr[i] === null) {
            if (hasNull == true) {
                continue;
            }
            hasNull = true;
        }

        if (type === "undefined") {
            if (hasUnderfined) {
                continue;
            }
            hasUnderfined = true;
        }

        var key = keyThunk(arr[i]);
        if (!dic[key]) {
            dic[key] = arr[i];
            result.push(arr[i]);
        }
    }
    return result;
}

var arr = [{ val: 1 }, { val: 1 }, { val: 3 }];

    return val.val

    return val.val;

    return val.val;
