< script >元素

  • async: 可选，立即下载脚本文件，但是不妨碍其他操作，只对外部脚本文件有效，在页面load事件前执行，但是可能在DOMContentLoaded之前或者之后执行
  • charset: 可选，表示通过src属性指定的代码的字符集
  • defer: 可选，表示脚本可以延迟到文档全部被解析和显示之后再执行，只对外部脚本文件有效，在DOMContentLoaded之前执行
  • src: 可选，表示要执行代码的外部文件
  • type: 可选，表示编写代码所使用的脚本语言的内容类型，也称MIME类型，服务器在传送JavaScript文件时使用的MIME类型通常是application/x-javascript，但在type中设置这个值可能导致脚本被忽略。application/javascript和application/ecmascript。约定俗成默认还是text/javascript
