JavaScript中每个函数内都能访问一个特别的变量arguments，MDN上给arguments的解释是：

arguments是一个类数组对象。代表传递给一个function的参数列表。

1. arguments length

arguments 是个类数组对象，其包含一个 length 属性，可以用 arguments.length 来获得传入函数的参数个数。

function func() {
    console.log("The number of parameters is " + arguments.length);
}
func();
func(1, 2);
func(1, 2, 3);
